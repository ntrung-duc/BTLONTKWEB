# 🚗 Secondhand Cars - Website Bán Ô Tô Cũ

Website thương mại điện tử chuyên về bán các chiếc ô tô đã qua sử dụng từ các hãng xe nổi tiếng thế giới.

## 📋 Mô tả Dự Án

**Secondhand Cars** là một nền tảng mua bán ô tô cũ trực tuyến, cung cấp một bộ sưu tập lớn các chiếc xe từ những thương hiệu hàng đầu như:
- 🏎️ Ford, Mercedes-Benz, Audi, Honda, Porsche, BMW, Toyota, Nissan

Trang web cung cấp giao diện thân thiện, cho phép khách hàng duyệt xem, tìm kiếm, và tìm hiểu thông tin chi tiết về các chiếc xe.

## ✨ Tính Năng Chính

- ✅ **Danh sách xe phân loại**: Sedan, Hatchback, SUV, Coupe
- ✅ **Lọc theo hãng xe**: Ford, Mercedes, Audi, Honda, Porsche, BMW, Toyota, Nissan
- ✅ **Slider quảng cáo**: Hiệu ứng video nền động cho các sản phẩm nổi bật
- ✅ **Tin tức cập nhật**: Phần tin tức hàng tuần về ngành ô tô
- ✅ **Responsive Design**: Giao diện tương thích với thiết bị di động
- ✅ **Hiệu ứng Hover**: Các hiệu ứng tương tác trên thương hiệu xe và tin tức
- ✅ **Liên hệ & Support**: Thông tin hotline, email, và đường dây nóng
- ✅ **Newsletter**: Đăng kí nhận bản tin cập nhật mới nhất

## 🛠️ Công Nghệ Sử Dụng

- **Frontend**: HTML5, CSS3, JavaScript
- **Library**: jQuery 3.7.1
- **Assets**: Hình ảnh, video, icon (favicon)
- **Responsive**: CSS Media Queries

## 🚀 Hướng Dẫn Sử Dụng

### Cách Chạy Dự Án

1. **Clone repository**
   ```bash
   git clone https://github.com/ntrung-duc/BTLONTKWEB.git
   cd BTLONTKWEB
   ```

2. **Mở file trong trình duyệt**
   - Mở `index.html` trực tiếp trong trình duyệt web
   - Hoặc sử dụng Live Server extension trong VS Code

3. **Duyệt web**
   - Trang chủ hiển thị slider quảng cáo
   - Bấm vào "Sản phẩm" để xem danh sách xe
   - Lọc theo loại xe hoặc hãng xe theo yêu cầu
   - Xem tin tức trong phần "Tin tức"
